# Library

Schema-first CRUD-API для markdown-борды знаний: заметки, теги,
связи между заметками и отдельная категория для документов,
сгенерированных ИИ и ожидающих канонизации человеком.

## Суть

Единственный источник правды — OpenAPI-спека. Из неё:

- генерируется HTTP-сервер на Go (oapi-codegen, strict-server) — для UI и людей;
- выводится набор MCP-инструментов — через них внутренний ИИ-ассистент
  получает типобезопасный read/write-доступ к борде.
